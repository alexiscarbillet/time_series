Neural network in time series


How to use neural network to predict the event in the futur? This project answers it by using time series. It only predicts the next value. It could predict many future values, like the 5 next ones. The code can be find here. The dataset used can be downloaded here.

Methods

First of all, the interesting features have been selected among 14 features available. Then, the model has been trained (20 epochs). Finally some predictions has been done. For more information about the methodology here.
